# 保有株トラッカー (PWA)

スマホのホーム画面に置いて全画面で使える株トラッカー。
保有株数・目標株数・現在株価から「残り株数」と「目標まで必要額」を計算します。

## 主な機能
- 残り株数・目標までの必要額・達成率の表示（単元株数も表示）
- 証券コードからStooq経由で株価を自動取得、30分ごとの自動更新
- 取得単価を入力すると評価損益、1株配当を入力すると目標達成時の年間配当を表示
- 予算を入力すると、必要額の少ない銘柄から優先的に何株買えるかを計算する予算シミュレーション
- 銘柄名・コードでの検索、達成率／必要額／名前での並び替え、手動の上下並べ替え
- 必要額合計の推移を表示するスパークライン（端末内履歴）
- データのJSON書き出し・読み込み（バックアップ・端末間移行）
- 目標達成時のブラウザ通知
- ライト／ダークのテーマ切替

## ファイル一式
- `index.html` … アプリ本体
- `manifest.webmanifest` … アプリ情報（名前・アイコン・全画面）
- `sw.js` … オフライン動作用 Service Worker
- `icon-192.png` / `icon-512.png` / `apple-touch-icon.png` … アイコン

※ 6ファイルすべてを **同じフォルダ** に置いてください。

## 公開（GitHub Pages・無料）
1. GitHubで新規リポジトリを作成（例: `kabu-tracker`）
2. 上記6ファイルをアップロード（ドラッグ&ドロップ可）
3. Settings → Pages → Branch を `main` / `/root` に設定して保存
4. 数十秒後に `https://<ユーザー名>.github.io/kabu-tracker/` が発行される

Netlify / Vercel にフォルダごとドロップしてもOK。

## ホーム画面に追加
- **iPhone (Safari)**: 発行URLを開く → 共有 → 「ホーム画面に追加」
- **Android (Chrome)**: URLを開く → メニュー → 「アプリをインストール」/「ホーム画面に追加」

追加後はアイコンから全画面で起動し、オフラインでも開けます。

## 株価の自動取得について
- データソースは Stooq（無料・キー不要）。日本株は4桁コード（例 `7867`）でOK、内部で `7867.jp` に変換します。
- 米国株などは `aapl.us` のように指定。
- Stooqがブラウザから直接取得できない場合は CORS プロキシ経由を自動で試します。それでも失敗したら **株価欄に手動入力** してください。
- 取得値は直近の終値ベースで、リアルタイムではありません。

## データ保存
入力内容は端末の `localStorage` に保存されます。ブラウザのデータを消去すると消えます（端末間の同期はありません）。
